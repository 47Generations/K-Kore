# K-Kore (DeX OS)
A modular plugin system for GTA San Andreas (MoonLoader v.027).

## Installation
1. Ensure you have **MoonLoader v.027.0-preview3** installed.
2. Place `CommandHandler.lua` and `InventoryHandler.lua` in your `/moonloader/` folder.
3. Place the `K-Kore` folder in your `/moonloader/` folder.

## Commands
- `/halojump` - Toggles Spartan-style jump physics.
- `/halo2` - Toggles Double Jump physics.
- `/v [ID]` - Spawns a vehicle.
- `/savetp [Name]` - Saves your current location.
